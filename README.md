# Terraform Modules

Abstraction and simplification of the infrastructure provisioning
using Terraform.

For examples, see [examples folder](examples).

Features:

1. AWS
    - VPC
    - EKS
    - EFS
    - ECR

More cloud providers and features will be added in the future.
